# Blog Template

A Next.js blog template with dynamic routing and content management.

## Features

- Dynamic blog post routes (`/blog/[slug]`)
- Tailwind CSS styling
- TypeScript support
- Responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

```
├── app/
│   ├── blog/[slug]/     # Dynamic blog post pages
│   ├── layout.tsx       # Root layout
│   └── page.tsx         # Homepage
├── components/          # React components
└── public/             # Static assets
```

## Customization

- Edit `app/page.tsx` to customize the homepage
- Add blog posts by creating new pages in `app/blog/[slug]/`
- Modify styles in `app/globals.css` and `tailwind.config.ts`
