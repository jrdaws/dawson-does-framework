# Dashboard Template

A Next.js admin dashboard template with sidebar navigation and settings.

## Features

- Admin dashboard layout
- Settings page
- Responsive sidebar navigation
- Tailwind CSS styling
- TypeScript support

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

```
├── app/
│   ├── settings/        # Settings page
│   ├── layout.tsx       # Root layout with sidebar
│   └── page.tsx         # Dashboard home
├── components/          # React components
└── public/             # Static assets
```

## Customization

- Edit `app/page.tsx` to customize the dashboard
- Add new pages in the `app/` directory
- Modify the sidebar in `app/layout.tsx`
- Update styles in `app/globals.css` and `tailwind.config.ts`
