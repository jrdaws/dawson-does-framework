STEP 3: GENERATION STRATEGY
---------------------------

For this project, use a mixed approach:

| Asset Type | Tool | Format | Why |
|------------|------|--------|-----|
| Icons | Stable Diffusion + SVG trace | SVG | Scalable, small |
| Illustrations | Stable Diffusion | WebP | Visual quality |
| Backgrounds | Stable Diffusion | WebP | Large areas |
| Gradients | CSS (no generation) | - | Code-based |
| Patterns | Stable Diffusion | SVG/PNG | Tileable |

### Cost-Optimized Approach
- Use Stable Diffusion for ALL drafts ($0.01/image)
- Only use DALL-E for final hero if needed
- Estimated total: $0.50-1.50 for complete redesign

STEP 5: OPTIMIZE ALL ASSETS
---------------------------

```bash
# For WebP images (requires imagemagick)
for img in raw/*.png; do
  convert "$img" -quality 85 "optimized/$(basename "$img" .png).webp"
done

# For SVGs (requires svgo)
for svg in raw/*.svg; do
  svgo "$svg" -o "svg/$(basename "$svg")"
done
```
