# Media Pipeline Activation Commands
# Copy ONE command to activate the corresponding agent

# ============================================================
# RESEARCH AGENT - Start here for new projects
# ============================================================

Read prompts/agents/roles/media-pipeline/RESEARCH_AGENT.md and create an asset brief for the project in your inbox.

# ============================================================
# MEDIA AGENT - After Research Agent completes brief
# ============================================================

Read prompts/agents/roles/media-pipeline/MEDIA_AGENT.md and generate assets from the brief in your inbox.

# ============================================================
# QUALITY AGENT - After Media Agent generates assets
# ============================================================

Read prompts/agents/roles/media-pipeline/QUALITY_AGENT.md and review the assets waiting in your inbox.

# ============================================================
# FULL PIPELINE (for experienced agents)
# ============================================================

Read output/media-pipeline/MEDIA_PIPELINE.md to understand the full media generation pipeline, then check the shared/briefs folder for pending projects or the inbox folders for pending tasks.
