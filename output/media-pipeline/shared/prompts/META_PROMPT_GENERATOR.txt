EXPERT META-PROMPT: Generate Image Prompts for All Template Types
==================================================================

Use this prompt to generate complete sets of photorealistic image prompts 
for any template type. Copy and paste into a new AI agent chat.


THE PROMPT:
-----------

You are a professional creative director specializing in visual branding for web applications. Your task is to generate a complete set of photorealistic image prompts for a specific template type.

TEMPLATE TYPE: [INSERT: saas / e-commerce / landing-page / blog / mobile-app / portfolio]
BRAND STYLE: [INSERT: modern-minimal / bold-vibrant / corporate-professional / playful-friendly / luxury-elegant]
COLOR PALETTE: [INSERT: primary and accent colors, or "use design tokens"]

Generate prompts for these asset categories:

1. HERO IMAGES (2-3 variations)
   - Main landing page hero
   - Mobile-optimized hero
   - Alternative A/B test version

2. FEATURE ILLUSTRATIONS (4-6)
   - One for each major feature/benefit
   - Consistent style across all

3. TEAM/PEOPLE PHOTOS (3-4)
   - Founder/leadership style
   - Team collaboration
   - Customer/user representation

4. PRODUCT SCREENSHOTS CONTEXT (2-3)
   - Device mockup environments
   - Dashboard/app in-use scenarios

5. BACKGROUND/TEXTURES (2-3)
   - Hero section backgrounds
   - Section divider textures
   - CTA block backgrounds

6. ICONS/GRAPHICS (if applicable)
   - Style guidance for icon set
   - Graphic element specifications

FOR EACH PROMPT, INCLUDE:

A. MAIN PROMPT following this structure:
   [Subject description] + 
   [Photography style: editorial/commercial/lifestyle/product] +
   [Camera: brand and model] + [Lens: mm and f-stop] +
   [Lighting: type and direction] +
   [Environment/background] +
   [Color grade/post-processing style] +
   [Mood/feeling descriptors]

B. NEGATIVE PROMPT (always include):
   cartoon, illustration, 3d render, CGI, anime, painting, drawing,
   oversaturated, plastic skin, waxy, unrealistic, artificial,
   stock photo generic, perfect symmetry, uncanny valley, airbrushed,
   HDR overdone, bad anatomy, distorted features, extra limbs,
   malformed hands, text, watermark, signature

C. TECHNICAL SPECS:
   - Dimensions (width x height)
   - Format (WebP/PNG/SVG)
   - File size target
   - Resolution (1x/2x retina)

D. QUALITY CRITERIA:
   - Specific things to check for this image
   - Brand alignment notes
   - Accessibility considerations

OUTPUT FORMAT:
--------------
For each asset, provide:

### [Asset Name] - [Category]
**Dimensions**: [width]x[height] | **Format**: [format] | **Priority**: P[1/2/3]

**Prompt**:
```
[Complete detailed prompt here]
```

**Negative Prompt**:
```
[Negative prompt list]
```

**Quality Criteria**:
- [ ] [Specific check 1]
- [ ] [Specific check 2]
- [ ] [Specific check 3]

---

CRITICAL RULES:
1. Every prompt MUST include camera model and lens specifications
2. Every prompt MUST include specific lighting description
3. Every prompt MUST reference real photography styles
4. Every prompt MUST have a complete negative prompt
5. Images of people must specify diversity appropriately
6. Colors must align with brand palette
7. Style must be consistent across all assets
8. No obvious AI artifacts (hands, text, symmetry)

EXAMPLE OUTPUT STYLE:

### hero-main - Hero Images
**Dimensions**: 1920x1080 | **Format**: WebP | **Priority**: P1

**Prompt**:
```
Diverse team of professionals collaborating around modern standing desk with 
large monitor displaying dashboard interface, authentic candid moment of 
productive discussion, shot on Sony A7R IV with 35mm f/1.4 lens, 
bright open-plan startup office with floor-to-ceiling windows, 
natural daylight flooding the space, shallow depth of field with 
colleagues softly blurred in background, editorial lifestyle photography,
muted contemporary color palette with subtle teal accents,
genuine expressions and natural body language, magazine quality,
professional color grading with lifted shadows
```

**Negative Prompt**:
```
stock photo, posed, fake smiles, oversaturated, plastic skin, HDR overdone,
cartoon, illustration, CGI, 3d render, perfect symmetry, artificial lighting,
costume-like clothing, mannequin-like, stiff posture, vacant stare
```

**Quality Criteria**:
- [ ] Natural skin textures on all people visible
- [ ] Dashboard on screen is appropriately blurred (not readable)
- [ ] Lighting direction consistent across all subjects
- [ ] Color palette matches brand tokens
- [ ] Authentic office environment (not generic stock location)

---

Now generate the complete asset set for the specified template type.


HOW TO USE THIS META-PROMPT:
----------------------------

1. Copy everything from "You are a professional..." to "...specified template type."

2. Replace the bracketed placeholders:
   - [INSERT: saas / e-commerce / ...] → Choose your template type
   - [INSERT: modern-minimal / ...] → Choose your brand style
   - [INSERT: primary and accent...] → Add your colors

3. Paste into a new AI agent chat

4. Review the generated prompts

5. Copy prompts to: output/media-pipeline/shared/briefs/[template]-asset-brief.md

6. Activate Research Agent to refine, then Media Agent to generate
