# UI Redesign Project - Agent Activation Commands
# ================================================
# 
# Copy ONE command to a new agent chat to activate that agent.
# Run agents IN ORDER: Research → Media → Quality
#
# Project: Dawson-Does-Framework UI Redesign
# Goal: Create user-friendly clone maintaining all features


# ============================================================
# STEP 1: RESEARCH AGENT
# ============================================================

Read output/media-pipeline/research-agent/inbox/PROJECT-framework-ui-redesign.txt and execute the UI redesign research task. Analyze the current website, define the new design direction, and create a comprehensive asset brief.


# ============================================================
# STEP 2: MEDIA AGENT (Run after Research completes)
# ============================================================

Read output/media-pipeline/media-agent/inbox/PROJECT-framework-ui-redesign.txt and generate all assets for the UI redesign. Follow the brief from Research Agent and ensure style consistency across all assets.


# ============================================================
# STEP 3: QUALITY AGENT (Run after Media completes)
# ============================================================

Read output/media-pipeline/quality-agent/inbox/PROJECT-framework-ui-redesign.txt and review all generated assets. Verify style consistency, quality standards, and approve or request revisions.


# ============================================================
# STEP 4: WEBSITE AGENT (Run after Quality approves)
# ============================================================

Read output/website-agent/inbox/media-framework-ui-redesign/IMPLEMENT.txt and implement the approved UI redesign assets. Update components and styles while maintaining all existing functionality.
