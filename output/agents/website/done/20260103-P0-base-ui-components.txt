## Implementation Requirements

1. **Use Tailwind CSS** - All styling via Tailwind classes
2. **TypeScript** - Full type definitions for all props
3. **Responsive** - Mobile-first, works on all screen sizes
4. **Dark theme compatible** - Use CSS variables or dark: variants
5. **No external dependencies** - Only React + Tailwind
6. **Accessible** - Proper ARIA labels, keyboard navigation
