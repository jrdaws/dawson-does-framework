# Sidebar Design Variants

Three different sidebar approaches using our Navy/Orange theme:

## Quick Open

```bash
open output/shared/design/demos/sidebar-variants/demo-1-floating-card.html
open output/shared/design/demos/sidebar-variants/demo-2-icon-rail.html
open output/shared/design/demos/sidebar-variants/demo-3-gradient-glass.html
```

## Demos

### Demo 1: Floating Card Sidebar
- **Style**: Sidebar as a floating card with padding from edges
- **Features**: Rounded corners, card shadow, elegant separation
- **Best for**: Premium/luxury feel, dashboard applications

### Demo 2: Icon Rail + Flyout  
- **Style**: Collapsed 80px icon rail that expands to 300px on hover
- **Features**: Tooltips in collapsed state, circular progress indicator
- **Best for**: Space-efficient layouts, power user applications

### Demo 3: Gradient Glassmorphism
- **Style**: Blur effects, gradient overlays, grouped sections
- **Features**: Shimmer animation, ambient glow, section grouping
- **Best for**: Modern SaaS, creative tools, futuristic aesthetics
